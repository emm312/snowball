Brief Description for strcat


It concatenates the string pointed to by source to the end of the string pointed to by destination.


Parameters


destination(&char)

- pointer to the destination array, which should contain a C string, and be large enough to contain the concatenated resulting string.


source(&char)

- C string to be appended. This should not overlap destination.

@return

&char - pointer to the destination string destination.
