Brief Description for memmove


(memmove) Copies count bytes from the object pointed to by src to the object pointed to by dest.


The result is a binary copy of the data.

Parameters


dest(c_obj)

- pointer to the object to copy to


src(c_obj)

- pointer to the object to copy from


count(c_int)

- number of bytes to copy

@note

(6) If the objects are potentially-overlapping or not TriviallyCopyable, use memmove or memcpy_s instead.

@return

c_obj - dest
