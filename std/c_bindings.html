Functions exported from std::c_bindings


malloc

 Allocates size bytes of uninitialized storage.

calloc

 Allocates memory for an array of num objects of size size and initializes all bytes in the allocated storage to zero.

realloc

 Reallocates the given area of memory. calloc() or realloc() and not yet freed with a call to free or realloc. Otherwise, the results are undefined.

free

 Deallocates the space previously allocated by malloc(), calloc() or realloc().

memcpy

 (memcpy) Copies count bytes from the object pointed to by src to the object pointed to by dest.

memmove

 (memmove) Copies count bytes from the object pointed to by src to the object pointed to by dest.

puts

 Writes every character from the null-terminated string str and one additional newline

putchar

 Writes a character to the standard output (stdout).

getchar

 Reads the next character from stdin.

printf

 Writes the string pointed by format to the standard output (stdout). If format includes format specifiers (subsequences beginning with %), the additional arguments following format are formatted and inserted in the resulting string replacing their respective specifiers.

snprintf

 Writes the string pointed by format to the character string pointed by str. If format includes format specifiers (subsequences beginning with %), the additional arguments following format are formatted and inserted in the resulting string replacing their respective specifiers.

exit

 Causes normal program termination to occur.

time

 It gets the current time.

strerror

 It returns a pointer to a null-terminated string that describes the error code passed in the argument errnum.

errno

 It returns the error number of the last error that occurred.
