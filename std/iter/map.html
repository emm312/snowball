Brief Description for map


It maps an iterator to a new iterator by applying a function to each element.



```
const myVector = /*{1, 2, 3, 4, 5}*\/; /// For example purposes only.
const mappedVector = map(myVector, (x) => x * 2);
// mappedVector is now {2, 4, 6, 8, 10}
```

Parameters

iterator

The iterator to map.


func

The function to apply to each element.

@return

A new iterator with the mapped elements.
