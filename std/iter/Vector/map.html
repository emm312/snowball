Brief Description for map


It maps the iterable to a new vector by applying a function to each element.



```
const myVector = /*{1, 2, 3, 4, 5}*\/; /// For example purposes only.
const mappedVector = myVector.map((x) => x * 2);
// mappedVector is now {2, 4, 6, 8, 10}
```

Parameters

func

The function to apply to each element.

@return

A new vector with the mapped elements.
