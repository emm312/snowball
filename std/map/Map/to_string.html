std::map::Map::to_string;

public func to_string<>() String { ... }


Returns a string representation of the map.

```
const myMap = new Map<string, number>();
myMap.set("foo", 42);
myMap.set("bar", 24);
const str = myMap.to_string(); // Returns "[("foo", 42), ("bar", 24)]"
```

@return

{String} A string representation of the map.
