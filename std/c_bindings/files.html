Brief Description for files


a namespace that contains functions for manipulating files.


@note


(1) All functions in this namespace are thread-safe.


Functions exported from std::c_bindings::files


fopen

 It opens the file whose name is specified in the parameter filename and associates it with a stream that can be identified in future operations by the FILE pointer returned.

fclose

 It closes the file associated with the stream and disassociates it.

fread

 It reads an array of count elements, each one with a size of size bytes, from the stream and stores them in the block of memory specified by ptr.

fwrite

 It writes an array of count elements, each one with a size of size bytes, from the block of memory pointed by ptr to the current position in the stream.

remove

 It removes a path from the filesystem.

access

 It checks if a file or directory access permissions can be checked.
