Brief Description for access


It checks if a file or directory access permissions can be checked.


On failure, a nonzero value is returned.

Parameters

filename(&char)

- C string containing the name of the file to be checked.


mode(c_int)

- This is the access mode to check for.

@return

If the file is successfully checked, a zero value is returned.
