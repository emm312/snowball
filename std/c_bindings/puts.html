Brief Description for puts


Writes every character from the null-terminated string str and one additional newline


Parameters

str(String)

This is the string to be written.

@func

fputc.

@note

(3) A typical cause of failure for puts is running out of space on the file system, when stdout is redirected to a file.

@return

(c_int) If successful, non-negative value is returned. On error, the function returns EOF.
