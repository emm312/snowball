Brief Description for calloc


Allocates memory for an array of num objects of size size and initializes all bytes in the allocated storage to zero.


The type of this pointer is always void*, which can be cast to the desired type of data pointer
in order to be dereferenceable.
but the returned pointer shall not be dereferenced.
and not any static storage.

Parameters


size(c_int)

- size of each object

@note

(3) calloc is thread-safe: it behaves as though only accessing the memory locations visible through its argument,

@return

c_obj - a pointer to the memory block allocated by the function.
